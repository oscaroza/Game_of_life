# importer le repo sur ton ordi :
```
cd
git clone https://github.com/oscaroza/Game_of_life.git
```

# Avant de coder, importer les modifications des autres :
```
cd Game_of_life
git pull
```

# exporter tes modifs :
```
cd Game_of_life
git pull
git add .
git commit -m "message expliquant la modification"
git push
```
